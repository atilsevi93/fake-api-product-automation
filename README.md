# Fake API CRUD Test Otomasyonu

## Açıklama
Bu proje, [https://fakeapi.platzi.com](https://fakeapi.platzi.com) üzerindeki ürünlerle ilgili CRUD işlemlerini test etmektedir.

## Teknolojiler
- Java 17
- Rest Assured
- TestNG
- Maven

## Senaryolar
- Ürün oluşturma (POST)
- Ürün listeleme ve alma (GET)
- Ürün güncelleme (PUT)
- Ürün silme (DELETE)

## Çalıştırma
```bash
mvn clean test
